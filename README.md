# 🎤 Assistente de Voz Multi-Idiomas com Whisper

Projeto desenvolvido no Google Colab que implementa um assistente de voz capaz de:
- 🎧 Captar áudio do usuário
- 🧠 Converter fala em texto (Speech-to-Text) com Whisper
- 🤖 Gerar uma resposta (simulada ou via API)
- 🔊 Converter texto em áudio (Text-to-Speech) com gTTS

---

## 🚀 Tecnologias utilizadas

- Python
- OpenAI Whisper
- gTTS (Google Text-to-Speech)
- JavaScript (MediaRecorder API - gravação no navegador)
- Google Colab

---

## ⚙️ Como funciona

1. O usuário grava um áudio pelo navegador  
2. O áudio é convertido em texto usando Whisper  
3. O sistema gera uma resposta (simulada ou via API OpenAI)  
4. A resposta é transformada em áudio com gTTS  

---

## 🧠 Observação importante

Devido à limitação de créditos da API da OpenAI, foi utilizada uma **resposta simulada**, mantendo a estrutura pronta para integração real com a API.

---

## ▶️ Como executar

1. Abra o notebook no Google Colab  
2. Execute as células em ordem  
3. Permita o uso do microfone  
4. Grave sua voz  
5. Ouça a resposta gerada  

---

Demonstrar a integração entre tecnologias de voz (Speech-to-Text e Text-to-Speech) com inteligência artificial, criando uma experiência interativa e acessível.
